# Kubernetes Cluster Setup

<!-- COMING SOON -->

This page will cover provisioning a Kubernetes cluster suitable for running SD-Core network functions and the srsRAN gNB as containerized workloads. The deployment will be driven through the custom Web UI. Topics will include cluster sizing, container runtime selection (containerd), kubeadm initialization, and networking prerequisites specific to telco workloads (low-latency, real-time scheduling, multiple network interfaces).

:::note
This section is pending completion of the custom deployment Web UI. The Kubernetes cluster provisioning workflow will be integrated into the UI-driven deployment process. Content will be added once the tooling is ready for use.
:::
