# Aether OnRamp Automation

<!-- COMING SOON -->

This page will cover using **Aether OnRamp** to automate the deployment of the complete Aether platform stack. OnRamp is an Ansible-based automation framework that provisions a Kubernetes cluster, deploys SD-Core, installs monitoring (Prometheus, Grafana), and configures the Runtime Operational Control (ROC) interface in a single automated workflow. It transforms the multi-step manual deployment process covered in the preceding pages into a repeatable, scriptable procedure suitable for production-like environments.

The custom Web UI used throughout this tutorial is built on top of the OnRamp automation, providing a graphical interface for the same underlying deployment steps.
